Jabber-Net

JabberClient

Namespace: jabber.client


Provides a component for clients to use to access the XMPP server. You can install this in your Toolbox, drop onto a form, a service, and so on. This class hooks into the OnProtocol event and calls the Connect() method.

OnLoginRequired
Signature: IEvent<ObjectHandler,unit>

Determines if SutoLogin is false, and if it is time to log in. This callback will receive the results of the IQ type=get in the jabber:iq:auth namespace. When login is completed, IsConnected property is set to true. If there is a login error, the FireAuthError() method is called.

OnRegisterInfo

Allows the user to enter registration requested information before sending to the XMPP server. WARNING: Make sure you do not return from this handler until the IQ is filled in. It is now safe to call UI elements, since this callback is now on the GUI thread if the InvokeControl is set.
