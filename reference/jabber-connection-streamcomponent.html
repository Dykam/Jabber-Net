Jabber-Net

StreamComponent

Namespace: jabber.connection


Manages the XmppStream as a component.

Constructors

ConstructorDescription
new()
Signature: unit -> unit

Instance members

Instance memberDescription
BeginIQ(iq, cb, cbArg)
Signature: (iq:IQ * cb:IqCB * cbArg:obj) -> unit

Does an asynchronous IQ call. If the from address hasn't been set, and an OverrideFrom has been set, the from address will be set to the value of OverrideFrom.

OnStreamChanged
Signature: IEvent<ObjectHandler,unit>

Informs the client that the XmppStream was changed. Often at design time, the object will be this StreamComponent.

OverrideFrom()
Signature: unit -> unit

Override the from address that will be stamped on outbound packets. Unless your server implemets XEP-193, you shouldn't use this for client connections.

Stream()
Signature: unit -> unit
Modifiers: abstract

Gets and sets the JabberClient or JabberService XMPP stream value.

Write(elem)
Signature: elem:XmlElement -> unit

Write the specified stanza to the stream. If the from address hasn't been set, and an OverrideFrom has been set, the from address will be set to the value of OverrideFrom.
