Jabber-Net

PubSubManager

Namespace: jabber.connection


Manages a set of publish-subscribe () subscriptions. The goal is to have a list of jid/node combinations, each of which is a singleton.

Signature: (service:JID * node:string * maxItems:int) -> PubSubNode

Subscribes to a publish-subscribe node.

Returns: The existing node will be returned if there is already a subscription. If the node does not exist, the PubSubNode object will be returned in a subscribing state.

OnError

Notifies the client that an error occurred. If this is set, it will be copied to each node that is created by the manager.
